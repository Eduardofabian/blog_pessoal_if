# 🚀 Blog Pessoal | Eduardo Fabian

Este repositório contém o código-fonte do meu blog e portfólio pessoal, desenvolvido como requisito de atividade acadêmica do curso de Análise e Desenvolvimento de Sistemas do Instituto Federal do Paraná (IFPR). 

O projeto apresenta minha trajetória como Engenheiro de Dados, minha formação acadêmica e meus hobbies através de um design minimalista e responsivo, focado na clareza da informação.

## 🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando os pilares clássicos do desenvolvimento web frontend:
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização com Flexbox, CSS Grid e variáveis customizadas. Implementação de um layout estilo "Dashboard" para os hobbies.
* **JavaScript:** Scripts básicos para interatividade e rolagem suave.

## 🤖 Uso de Inteligência Artificial
Atendendo às diretrizes da atividade, este projeto foi construído em colaboração com uma Inteligência Artificial.

* **IA Utilizada:** Gemini (Google).
* **Como a IA auxiliou no projeto:**
    * Planejamento da arquitetura de arquivos e pastas.
    * Tradução da minha história e currículo para a marcação HTML.
    * Geração do código CSS inspirado no design system minimalista da Indicium, convertendo ideias visuais (como cards estilo KPI) em código funcional e responsivo.
    * Resolução de avisos de compatibilidade (linter) no CSS.

## ⚙️ Como visualizar
Como é um projeto de site estático, não é necessária nenhuma instalação complexa:
1. Faça o clone deste repositório: `git clone https://github.com/SEU-USUARIO/blog_pessoal_if.git`
2. Abra a pasta do projeto e dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador de preferência.
